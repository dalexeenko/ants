## What is OpenMgr?

OpenMgr is a self-hosted platform for creating, managing, and running AI coding agents. It gives you full control over your AI development infrastructure without depending on external services.

<CardGrid stagger>
  <Card title="Self-Hosted Server" icon="server">

    npx, Docker, or bare metal — your infrastructure, your data.
  </Card>
  <Card title="Multi-Platform Apps" icon="laptop">
    Electron desktop app, React Native mobile app, and a web UI. Manage your

  </Card>
  <Card title="14+ LLM Providers" icon="puzzle">

## Key Features

- **Project Management** — Create and manage AI coding projects with dedicated workspaces
- **Session Management** — Create agent sessions, send prompts, stream responses via SSE
- **File Operations** — Browse, read, write, and manage project files through the API
- **Terminal Sessions** — WebSocket-based terminal I/O within projects
- **Task Scheduling** — Scheduled and recurring agent tasks
- **Messaging Integrations** — Connect agents to Slack (with Discord, Telegram, and more planned)
- **Human-in-the-Loop** — Approval workflows for agent actions
- **Vector Memory** — Embedding-based knowledge base for agents
- **Multi-User Mode** — RBAC with admin, operator, and viewer roles
- **Docker Support** — Full and lite image variants with Docker Compose support

## Quick Start

The fastest way to get running:

```bash
npx @openmgr/server
```

Or with Docker:

```bash
docker run -p 6647:6647 \
  -v openmgr-data:/data \
  -v openmgr-workspaces:/workspaces \
  -e OPENMGR_ENCRYPTION_KEY=$(openssl rand -base64 32) \
  openmgr/server
```

See the [Quickstart guide](/getting-started/quickstart/) for full setup instructions.
